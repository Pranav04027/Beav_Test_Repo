# Beav Test App

A dummy application used for testing the Beav automated issue fixer.

## Running Tests

```bash
npm test
```

## Modules

- **calculator.js** - Basic math operations (add, subtract, multiply, divide, average, factorial, power, absolute)
- **strings.js** - String manipulation (capitalize, reverse, countVowels, isPalindrome, truncate, wordCount)
- **arrays.js** - Array utilities (unique, flatten, chunk, removeDuplicates, sum, max, min)
